libfcd [![Build Status](https://secure.travis-ci.org/jrcutler/libfcd.png)](http://travis-ci.org/jrcutler/libfcd)
======

libfcd is an interface library for the FUNcube dongle.

Dependencies
------------

* Linux: `libusb-1.0`
* FreeBSD: `libusb-1.0`

Building
--------

### Release tarball

    ./configure
    make
    sudo make install

### From git

    ./bootstrap
    ./configure
    make
    sudo make install
